# Proyecto analitico AquaLimpia S. A.

## Problema

AquaLimpia S. A. presenta incumplimientos intermitentes en parametros criticos del efluente tratado, especialmente DBO y eficiencia del tratamiento. Como no existe un patron evidente, se requiere un analisis reproducible que permita apoyar decisiones operacionales, ambientales y gerenciales.

## Solucion analitica propuesta

La problematica se aborda mediante un flujo de analisis en Python que:

1. Lee el dataset oficial en Excel.
2. Limpia y tipifica las variables.
3. Calcula indicadores de desempeno ambiental y operacional.
4. Genera archivos separados para Operaciones y Gestion Ambiental.
5. Construye un dashboard exploratorio en HTML.
6. Produce un informe final en Markdown.

Esta solucion se justifica porque permite convertir datos historicos en evidencia para la toma de decisiones. Operaciones puede detectar alertas y revisar eficiencia del proceso; Gestion Ambiental puede respaldar reportes normativos; y gerencia puede priorizar inversiones o ajustes operacionales en las plantas con menor desempeno.

## Indicadores incluidos

- Cumplimiento normativo.
- DBO de entrada y salida.
- Eficiencia de remocion de DBO.
- Carga de DBO de entrada y salida.
- Energia por metro cubico tratado.
- Lodos generados.
- Alerta operativa.
